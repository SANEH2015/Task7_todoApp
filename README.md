<h1>Task7 Todo App</h1>
## Run Locally

Clone the project

```bash
  git clone https://github.com/SANEH2015/Task7_todoApp.git
```
Go to the project directory
```bash
  cd frondend 
```
Install dependencies
```bash
  npm install
```
Start the react server
```bash
  npm run start
```
## Tech Stack
**Client:** React JS, HTML5, JavaScript, CSS3
